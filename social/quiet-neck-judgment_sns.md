# SNS投稿文 — 首が静かになると、判断が変わる

## X投稿（3種類）

### ① 問いかけ型
```
「あのとき、なぜあんな判断をしたんだろう」

後から気づく後悔の多くに、
「あのとき、首が緊張していた」という事実が隠れているかもしれない。

思考は、頭だけではなく身体ごとしている。

#身体感覚 #判断 #首こり
```

### ② 学び型
```
首の慢性緊張は、思考を「緊急モード」に傾けやすくする。

素早い判断、短期的な視点、問題を排除しようとする思考——
それは緊急時に適したモードだが、
日常の多くの場面では必要ない。

首が静かになると、思考の入り口が変わる。

#神経系 #身体感覚 #セルフケア
```

### ③ 感情型
```
首の緊張が取れたとき、
さっきと同じことを考えているのに
なんか違う、と感じたことはありますか。

あれは気のせいじゃなかったかもしれない。

身体の状態が変わると、
思考のモードが変わることがある。

#身体の話 #首 #思考
```
