# SNS投稿文 — 神経ゲインとは何か

## X投稿（3種類）

### ① 問いかけ型
```
休んだのに、疲れが残っている。

7時間寝た。何もしていない週末だった。
なのに月曜の朝、首と肩はもう張っている。

体力がないのではなく、
神経系がずっとフル回転しているのかもしれない。

#神経ゲイン #疲労 #身体感覚
```

### ② 学び型
```
「神経ゲイン」＝神経系の信号の増幅率。

ゲインが高いとき、身体は通常より強く反応する。
音が気になる。光がまぶしい。何もしていないのに首が張る。

「繊細すぎる」のではなく、
神経系が高い感度で動き続けているのだ。

#神経ゲイン #神経系 #セルフケア
```

### ③ 感情型
```
「もっと頑張れば回復できる」と思ってきた。

でもアクセルを踏み続けたまま
エンジンを大きくしても、疲れは変わらない。

疲れへのアプローチが変わった話。

#身体感覚 #疲労 #神経系
```

## Instagramリール台本

**冒頭フック（0〜3秒）**
> 休んだのに疲れが残る理由、知ってますか？

**本題（3〜45秒）**
> スライド1: 神経ゲインとは、神経系の信号の「増幅率」のこと
>
> スライド2: ゲインが高いとき——音が刺さる、光がまぶしい、首と肩が何もしていないのに張る
>
> スライド3: 慢性的なストレスや集中作業が続くと、神経系はゲインを下げるタイミングを失う
>
> スライド4: 「リラックスしよう」と思っても効かないのは、意志ではなく身体の信号で下げるものだから
>
> スライド5: ゆっくり息を吐く→首の力みを感じる→もう一度吐く。それが「安全シグナル」になる

**CTA（45〜50秒）**
> 詳しくはプロフのnoteへ。「神経ゲイン基礎講座」シリーズ第1回。

## note導入文（150字以内）

> 休んだのに疲れが残る。その正体は体力ではなく、神経系の「増幅率」にあるかもしれない。神経ゲインという概念を知ると、疲れへの向き合い方が変わる。
