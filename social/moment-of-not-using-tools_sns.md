```
使わなくなった後も、身体の中に残っていく。

#身体感覚 #セルフケア #Tri-Aid
```

```

「いざとなれば戻れる場所」として
そこにあるのかもしれない、と気づいた話。

#セルフケア #身体の話 #道具
```
